# Web app to Android app with Capacitor

### Dependencies
 - nodejs
 - npx
 - yarn
 - capacitor/cli and capacitor/core
 - cordova-res
 - gradle
 - jre
 - jdk
 - android-sdk
 - jarsigner
 - zipalign

### Building android app
```bash
gradle assemble
gradle assembleDebug
gradle assembleRelease
```
