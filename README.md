### How to install NODE and NPM using NodeSource
add source
```bash
curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
```
install from apt
```bash
sudo apt install nodejs
```
build tools
```bash
sudo apt install build-essential
```
### How to install YARN
 ```bash
 curl -o- -L https://yarnpkg.com/install.sh | bash
 ```
